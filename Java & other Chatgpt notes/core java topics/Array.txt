 1. What is an array in Java?

Answer:
An array in Java is a container object that holds a fixed number of values of a single data type. Arrays are used to store multiple values in a single variable.

Syntax to declare an array:

java
int[] arr = new int[5];  // Array of 5 integers


Each element of an array is accessed by an index, starting from 0.

Example:

java
int[] arr = {1, 2, 3, 4, 5};
System.out.println(arr[0]);  // Output: 1

 3. What is the default value of an array element in Java?

Answer:
The default value of an array element depends on the type of array:

* Object references: Default value is null.

Example:

java
int[] arr = new int[5];  // Array of 5 integers

 6. What is the difference between an array and an ArrayList in Java?

Answer:

* Array:

  * Fixed size.
  * Can store elements of a single type (primitive or object).
  * Does not provide built-in methods for resizing or adding/removing elements.
  * Syntax for initialization:

    java
    int[] arr = {1, 2, 3};
    

* ArrayList:

  * Resizable (can grow and shrink dynamically).
  * Only stores objects (no primitive types).
  * Provides several methods (e.g., add(), remove(), size()).
  * Syntax for initialization:

    java
    ArrayList<Integer> list = new ArrayList<>();
    list.add(1);
    list.add(2);
    list.add(3);

 9. What is the difference between shallow copy and deep copy of an array?

Answer:

* Shallow Copy: Copies the references of the objects in the array. Any change to one object in the array will reflect in the copied array.

  Example (shallow copy using clone()):

  java
  int[] arr1 = {1, 2, 3};
  int[] arr2 = arr1.clone();  // Shallow copy
  

* Deep Copy: Creates a new array and also copies the objects themselves, not just their references. Changes in one array will not affect the other array.

  Example (deep copy):

  java
  int[][] arr1 = {{1, 2}, {3, 4}};
  int[][] arr2 = new int[arr1.length][];
  for (int i = 0; i < arr1.length; i++) {
      arr2[i] = arr1[i].clone();  // Deep copy for each sub-array
  }

 10. How do you sort an array in Java?

Answer:
You can sort an array in Java using the Arrays.sort() method.

Example:

java
int[] arr = {5, 3, 8, 1, 2};
Arrays.sort(arr);
System.out.println(Arrays.toString(arr));  // Output: [1, 2, 3, 5, 8]


For descending order, you can use a custom comparator or reverse the array after sorting:

java
Integer[] arr = {5, 3, 8, 1, 2};
Arrays.sort(arr, Collections.reverseOrder());
System.out.println(Arrays.toString(arr));  // Output: [8, 5, 3, 2, 1]

 11. What is the time complexity of sorting an array in Java?

Answer:
The time complexity of sorting an array using Arrays.sort() in Java depends on the type of array:

* Primitive arrays (like int[], char[]): O(n log n) using Dual-Pivot Quicksort.
* Object arrays (like Integer[], String[]): O(n log n) using TimSort, which is a hybrid sorting algorithm derived from merge sort and insertion sort.

 7. How do you reverse an array in Java?

Answer:
You can reverse an array by swapping elements from the beginning with the elements from the end.

Example:

java
int[] arr = {1, 2, 3, 4, 5};
int start = 0, end = arr.length - 1;
while (start < end) {
    int temp = arr[start];
    arr[start] = arr[end];
    arr[end] = temp;
    start++;
    end--;
}
System.out.println(Arrays.toString(arr));  // Output: [5, 4, 3, 2, 1]

 8. What is the difference between == and equals() when comparing arrays?

Answer:

* ==: Checks if two references point to the same array object in memory.
* equals(): By default, checks if two arrays are referentially equal (i.e., points to the same object). However, it doesn’t compare the contents of the arrays. To compare contents, you would use Arrays.equals().

Example:

java
int[] arr1 = {1, 2, 3};
int[] arr2 = {1, 2, 3};
System.out.println(arr1 == arr2);  // Output: false (different objects)
System.out.println(Arrays.equals(arr1, arr2));  // Output: true (same content)

 15. What is the purpose of the clone() method in arrays?

Answer:
The clone() method creates a shallow copy of the array. The new array will have the same elements as the original array, but changes to the cloned array do not affect the original array (for primitive types).

Example:

java
int[] arr = {1, 2, 3};
int[] arrClone = arr.clone();
System.out.println(Arrays.toString(arrClone));  // Output: [1, 2, 3]
